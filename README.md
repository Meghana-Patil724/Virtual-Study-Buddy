# Study Buddy

## Overview

Study Buddy is a web-based student productivity platform designed to help students manage their academic activities efficiently. The system provides a centralized environment for organizing study materials, tracking assignments, and improving time management.

## Features

* User-friendly interface
* Study material organization
* Assignment and task tracking
* Academic resource management
* Student-focused productivity tools
* Responsive web design

## Technologies Used

* HTML
* CSS
* PHP
* MySQL
* JavaScript

## Project Objectives

The main objective of Study Buddy is to help students:

* Organize academic resources effectively
* Improve study planning and scheduling
* Track assignments and deadlines
* Enhance productivity and learning efficiency

## System Modules

### User Management

* User registration and login
* Secure access to resources

### Study Resources

* Store and access study materials
* Organize notes and academic content

### Task Management

* Track assignments and deadlines
* Monitor academic progress

## Installation

1. Download or clone the repository.
2. Place the project folder in your web server directory.
3. Configure the database connection.
4. Import the database into MySQL.
5. Run the project using XAMPP or a similar local server.
